# 门票助手应用

## 安装说明

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 设置环境变量
确保设置以下环境变量：
```bash
export DASHSCOPE_API_KEY=你的_api_key
```

### 3. 运行应用
```bash
python main.py
```

## 功能
- 门票订单数据查询
- SQL执行和可视化
- Web界面交互

## 注意事项
- 需要有效的数据库连接
- 需要有效的DashScope API密钥
